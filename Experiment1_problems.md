# Edabit Problems for Experiment 1

This appendix summary briefly explains the Edabit programming problems selected for Experiment 1.

## Hard Problems

### The Snake – Area Filling
A grid-based problem involving a snake or path that occupies/fills space. The task is generally focused on tracking movement or coverage across a defined area and determining the final filled area or occupied region.

### The Karaca’s Encryption Algorithm
A string manipulation and encryption problem. The solution typically involves transforming a word by reversing it, replacing vowels with specific numeric values, and adding a fixed suffix.

### Convert to Hex
A conversion problem where text or characters are converted into hexadecimal form. This mainly tests string handling, character encoding, and formatting.

### Flipping Bits
A bitwise manipulation problem. The task involves flipping binary bits, often within a fixed-width integer representation, and returning the resulting value.

## Very Hard Problems

### Atbash Cipher
A substitution cipher problem where each letter is replaced with its opposite letter in the alphabet. For example, `a` maps to `z`, `b` maps to `y`, and so on.

### Advanced List Sort
A sorting problem involving more complex ordering rules than a basic ascending or descending sort. The task usually requires handling structured data or applying a custom sorting condition.

### Recursion: Palindrome Phrase
A recursion-based string problem. The task is to determine whether a phrase is a palindrome, usually while ignoring spaces, punctuation, and letter case.

### Prime Factorization of an Integer
A number theory problem requiring an integer to be broken down into its prime factors. This tests loops, divisibility checks, and efficient factorisation logic.

## Expert Problems

### Level Order Traversal
A tree traversal problem. The task requires visiting nodes level by level, usually from left to right, using a breadth-first search approach.

### Binary Search Tree
A data structures problem involving binary search tree rules. The task may involve constructing, validating, searching, or traversing a tree where left-side values are smaller and right-side values are larger.

### Hash Juggling
A problem involving hash-based data handling. The task generally requires using dictionaries, hash maps, or key-value relationships to track, rearrange, or process values efficiently.

### Keyword Cipher
A substitution cipher problem that uses a keyword to build a custom alphabet. The task involves encoding or decoding text using the generated cipher alphabet.
